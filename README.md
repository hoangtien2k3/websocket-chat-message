## Spring boot websocket chat message demo

## Clone source:
```text
https://github.com/hoangtien2k3qx1/websocket-chat-message.git
```

## Run file (open terminal)
```text
mvn spring-boot:run
```

## Link url localhost:8080
```text
http://localhost:8080/index.html
```
